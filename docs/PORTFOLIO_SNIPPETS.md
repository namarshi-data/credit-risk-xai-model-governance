## LinkedIn project description

I built an end-to-end retail credit risk analytics project that simulates how a Canadian lender could identify elevated borrower default risk, monitor portfolio quality, explain model decisions, and document governance controls. The workflow covers multi-sheet Excel ingestion, data quality review, portfolio EDA, leakage-controlled feature engineering, XGBoost and Random Forest modelling, threshold selection using review-rate and business-cost constraints, SHAP explainability, local explanations, counterfactual diagnostics, and model governance documentation.
